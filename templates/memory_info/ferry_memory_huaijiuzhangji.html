俄罗斯方块

俄罗斯方块(Tetris， 俄文:Тетрис)是一款电视游戏机和掌上游戏机游戏，它由俄罗斯人阿列克谢·帕基特诺夫发明，故得此名。 俄罗斯方块的基本规则是移动、旋转和摆放游戏自动输出的各种方块，使之排列成完整的一行或多行并且消除得分。 2014年2月22日，英国心理学家发现，减肥者通过玩俄罗斯方块，可以降低对富含脂肪的食物和零食的心理需求;尝试戒烟者的烟瘾也能得到遏制。

坦克大战

地方多辆坦克，自己发射的炮弹先打到敌方某坦克，则敌方该坦克消灭。

贪吃蛇

吃的越多，蛇的身体越长，当头部碰到尾部则失败。

射击和填补射击

射击，无脑左右扫射。而填补射击就有点难度了，跟俄罗斯方块差不多。
